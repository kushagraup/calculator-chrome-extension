## How to use
1.  Go to extantion list ([chrome://extensions/](chrome://extensions/))
2.  Enable "Developer mode"
3.  Press "Load unpacked"
4.  Select folder with extantion files (manifest, js, css, html)

## Techstack 
Written in JavaScript, HTML and CSS
